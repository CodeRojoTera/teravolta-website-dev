## Vision
To polish the existing Teravolta web application, stabilizing user flows, connecting disjointed sections, and resolving technical errors to ensure a seamless experience for Customers, Technicians, and Administrators.

## Goals
1.  **Stabilize User Flows:** Identify and fix logic errors in core workflows (Onboarding, Quoting, Service Requests).
2.  **Unify Sections:** Ensure seamless navigation and state continuity between different portal areas (e.g., Onboarding -> Dashboard).
3.  **Polish UI/UX:** Refine visual consistency and feedback mechanisms across the application.

## Non-Goals (Out of Scope)
-   Building new major features (unless required to fix a flow).
-   Complete re-write of the backend.

## Users
-   **Customers:** Seeking services, viewing quotes, managing projects.
-   **Technicians:** Onboarding, viewing schedules, managing jobs.
-   **Admins:** Overseeing users, requests, and system status.

## Success Criteria
- [ ] Technician onboarding flow completes without redirection errors.
- [ ] Customer service request flow connects correctly to Admin dashboard.
- [ ] "Countless errors" in flow logic are identified and resolved.
- [ ] Navigation between portal sections is intuitive and unbroken.
